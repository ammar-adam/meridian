# Meridian teaser (Remotion)

30-second Quiet Capital hype cut for investor outreach / LinkedIn.

## Commands

```bash
cd remotion
npm install
npm run studio    # preview
npm run render    # writes out/meridian-teaser.mp4
```

## Beats (~30s @ 30fps)

1. Brand mark  
2. “Community deal flow, with receipts.”  
3. Deal Flow product frame (staggered rows)  
4. Proof stats (live ballpark numbers from `src/brand.ts`)  
5. Close CTA  

Update `stats` in `src/brand.ts` if prod numbers move before you ship the cut.
